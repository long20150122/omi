## Switch

Switch toggle the state of a single setting on or off. They are the preferred way to adjust settings on mobile.

## Usage

```jsx
<m-switch label='Label1'></m-switch>

<m-switch label='Label2'></m-switch>

<m-switch label='Label3'></m-switch>
```

## API

### Props

```jsx
{
  label?: string,
  disabled?: boolean,
  checked?: boolean,
  value: string
}
```
