## Switch

切换打开或关闭单个设置的状态。 它们是调整移动设置的首选方式。

## 使用

```jsx
<m-switch label='Label1'></m-switch>
<m-switch label='Label2'></m-switch>
<m-switch label='Label3'></m-switch>
```
